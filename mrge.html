Capstone Project

Mixed Reality Game Engine

A web-based, location-based mixed-reality escape room system developed for AI3D PLT from Jul 2025 to Jul 2026.

Overview

This project supports two roles: admin and player.

  • Players follow location-based clues and scan real-world objects.
  • Players unlock digital content, answer questions, earn points, and track progress.
  • Admins configure rooms, storylines, questions, object detection, and player progress.
