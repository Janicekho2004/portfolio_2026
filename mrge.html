Capstone Project

Mixed Reality Game Engine

Final-Year Project 2025–2026 Project Manager & Developer

A location-based mixed-reality escape room platform that combines real-world exploration, object scanning and interactive digital gameplay.

Project Overview

The platform supports two main user roles: administrator and player.

  • Players follow location-based clues and scan real-world objects to progress through the game.
  • Correct scans unlock digital assets, questions and interactive activities that allow players to earn points.
  • Administrators configure rooms, storylines, routes, questions, assets, object-detection models and player progress.
